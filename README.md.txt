# FPGA Implementation of Radix-16 Booth Multiplier with Error Detection

## Overview

Designed and implemented a high-speed signed Radix-16 Booth Multiplier using Verilog HDL and Xilinx Vivado.

The project focuses on:
- Fast multiplication architecture
- Booth encoding optimization
- FPGA implementation
- On-chip debugging using ILA
- Simulation and synthesis verification

---

## Features

- Signed 16-bit multiplication
- Radix-16 Booth Encoding
- FSM-based sequential design
- FPGA validation
- Debounce and edge-detection logic
- Integrated waveform analysis

---

## Tools Used

- Verilog HDL
- Xilinx Vivado
- FPGA Design Flow

---
